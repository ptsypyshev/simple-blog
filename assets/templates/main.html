Сделано наспех, реализованы только CRUD-методы работы с объектами User, Post и Comment.

Доступный функционал:

Users - Пользователи

Модель данных

type User struct {
Id int `json:"id"`
Username string `json:"username"`
Password string `json:"password"`
FirstName string `json:"first_name"`
LastName string `json:"last_name"`
Email string `json:"email"`
IsActive bool `json:"is_active"`
}

Методы (path /users/)

  • GET - Получить данные пользователя

    curl -X GET http://localhost:8080/users/5

  • POST - Создать нового пользователя

    curl -X POST http://localhost:8080/users/ -H 'Content-Type: application/json' -d '{"username":"tester","password":"test","first_name":"First","last_name":"Last","email":"tester@example.loc"}'

  • PUT - Обновить данные пользователя

    curl -X PUT http://localhost:8080/users/ -H 'Content-Type: application/json' -d '{"id":7,"username":"Updated","is_active":true}'

  • DELETE - Удалить пользователя

    curl -X DELETE http://localhost:8080/users/5

Posts - посты блога

Модель данных

type Post struct {
Id int `json:"id"`
Title string `json:"title"`
Body string `json:"body"`
UserId int `json:"user_id"`
}

Методы (path /posts/)

  • GET - Получить пост

    curl -X GET http://localhost:8080/posts/7

  • POST - Создать нового поста

    curl -X POST http://localhost:8080/posts/ -H 'Content-Type: application/json' -d '{"title":"NewPost","body":"New Content","user_id":7}'

  • PUT - Обновить данные поста

    curl -X PUT http://localhost:8080/posts/ -H 'Content-Type: application/json' -d '{"id":5,"title":"Updated1","body":"Updated Content"}'

  • DELETE - Удалить пост

    curl -X DELETE http://localhost:8080/posts/7

Comments - Комментарии

Модель данных

type Comment struct {
Id int `json:"id"`
Date time.Time `json:"date"`
Body string `json:"body"`
UserId int `json:"user_id"`
PostId int `json:"post_id"`
}

Методы (path /comments/)

  • GET - Получить комментарий

    curl -X GET http://localhost:8080/comments/7

  • POST - Создать нового комментария

    curl -X POST http://localhost:8080/comments/ -H 'Content-Type: application/json' -d '{"body":"New Content","user_id":4,"post_id":2}'

  • PUT - Обновить комментарий

    curl -X PUT http://localhost:8080/comments/ -H 'Content-Type: application/json' -d '{"id":5,"body":"Updated Content"}'

    curl -X PUT http://localhost:8080/comments/ -H 'Content-Type: application/json' -d '{"id":6,"body":"Updated Comment","user_id":1}'

  • DELETE - Удалить комментарий

    curl -X DELETE http://localhost:8080/comments/7
